Prérequis

  • Un Launcher capable de lancer Minecraft: Java
  • Version du jeu : 1.21.4
  • Connexion Internet stable (VPN Interdit)
  • Un compte Discord
  • IP du Serveur : play.true-vanilla.fr

Étape 1 : Lier votre compte Minecraft à votre compte Discord

Rejoignez le Discord pour pouvoir lier votre compte Minecraft à votre compte Discord.

Rejoindre le Discord

Rejoignez le serveur Minecraft pour recevoir votre code qui vous permettra de lier vos comptes.

Envoyez le code au bot du serveur Discord "True Vanilla V3".

Étape 2 : Se connecter au serveur

Une fois vos comptes liés, créez un compte sur le serveur en utilisant :

/register motdepasse motdepasse

Connectez-vous en utilisant :

/login motdepasse ⠀
⚠️ Attention ! Remplacez "motdepasse" par votre mot de passe personnel (minimum 4 caractères)
